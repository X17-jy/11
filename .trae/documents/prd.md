# 数据分析在线教育平台 PRD

## 1. 产品概述

数据分析在线教育平台是一个面向商务数据分析与应用专业学生的免费学习平台。无需注册登录，打开即可学习。平台提供 Python 数据分析相关课程和实战项目，帮助学生掌握 Pandas、数据可视化、商务数据分析等核心技能。

## 2. 核心功能

### 2.1 页面结构

| 页面 | 功能描述 |
|------|----------|
| 首页 | 展示平台介绍、课程推荐、项目推荐 |
| 课程页 | 展示 Python 数据分析课程列表 |
| 项目页 | 展示商务数据分析实战项目 |

### 2.2 页面详情

| 模块 | 功能描述 |

## 3. 内容规划

### 3.1 课程内容

| 课程名称 | 描述 | 标签 |
|----------|------|------|
| Python数据分析基础 | 学习Python数据分析的基本概念和方法 | Pandas / 入门 |
| Pandas数据处理实战 | 掌握Pandas库进行数据清洗、转换和分析 | Pandas / 进阶 |
| 数据可视化与图表制作 | 使用Matplotlib和Seaborn创建专业图表 | 可视化 / 图表 |
| 商务数据分析实战 | 运用数据分析技能解决真实商务问题 | 商务分析 / 实战 |

### 3.2 项目内容

| 项目名称 | 描述 | 数据集 | 任务 |
|----------|------|--------|------|
| 电商销售数据分析 | 分析电商平台销售数据，发现销售规律 | 订单数据、商品数据 | 数据清洗、趋势分析、销售排行 |
| 客户流失预测分析 | 预测客户流失风险，制定挽留策略 | 客户行为数据 | 特征工程、模型构建、风险评估 |
| 市场营销效果分析 | 评估营销活动效果，优化投放策略 | 营销活动数据 | ROI计算、渠道分析、效果评估 |

## 5. 设计风格

### 5.1 色彩方案
- 主色调：蓝色 (#0d6efd, #2563eb)
- 辅助色：白色 (#ffffff)
- 背景色：浅灰 (#f8f9fa)
- 强调色：深灰 (#495057)
- 文字色：深灰 (#212529)

### 5.2 字体
- 标题：思源黑体 / Noto Sans SC
- 正文：思源黑体 / Noto Sans SC

### 5.3 布局
- 响应式设计，桌面优先
- 固定顶部导航栏
- 卡片式内容布局
- 网格系统：Bootstrap 12栅格

### 5.4 动画
- 卡片悬停轻微上浮效果
- 页面加载淡入效果
