# 商务数据分析在线教育平台 Spec

## Why
面向商务数据分析与应用专业学生，提供一个无需登录、开箱即用的在线教育平台，集中展示课程与项目实践内容，降低学习入口门槛，帮助学生快速了解平台能力并进入学习状态。

## What Changes
- 新增 Flask Web 项目结构（单文件应用 + 模板 + 静态资源）
- 新增三个核心页面：首页（`/`）、课程页（`/courses`）、项目页（`/projects`）
- 新增项目详情页（`/projects/<id>`）展示完整项目说明
- 新增课程详情页（`/courses/<id>`）展示课程详情
- 新增统一导航栏与页脚
- 使用蓝白灰三色系、清爽专业风格
- 内置示例数据：4 门课程 + 4 个商务数据分析项目
- **BREAKING** 无：项目为新增独立应用，不影响其他系统

## Impact
- Affected specs: 无（全新项目）
- Affected code: 整个 `/workspace` 目录（项目根目录）
- 技术栈：Python 3.x + Flask + Jinja2 模板 + 原生 CSS（无前端构建工具）
- 运行依赖：仅 `flask` 一个第三方包

## ADDED Requirements

### Requirement: 平台首页
系统 SHALL 在 `/` 路由提供平台首页，包含：顶部 Banner 区域（平台介绍与 CTA 按钮）、课程推荐区（展示 3 门精选课程卡片）、项目推荐区（展示 3 个推荐项目卡片）、平台特色数据展示区。

#### Scenario: 用户访问首页
- **WHEN** 用户在浏览器中访问 `/`
- **THEN** 页面渲染平台介绍、推荐课程列表与推荐项目列表，所有数据来自后端字典配置

### Requirement: 课程列表页
系统 SHALL 在 `/courses` 路由提供课程列表页，以网格卡片形式展示所有 Python 数据分析相关课程，每张课程卡片需包含：课程名称、分类标签（如 Pandas / 数据可视化 / 商务数据分析）、难度等级、课程简介。

#### Scenario: 用户浏览课程列表
- **WHEN** 用户访问 `/courses`
- **THEN** 页面以卡片网格形式展示全部课程，每门课程可点击进入详情页

### Requirement: 课程详情页
系统 SHALL 在 `/courses/<course_id>` 路由提供单门课程的详细信息，包括：课程目标、章节大纲、适合人群、学习收获。

#### Scenario: 用户点击某门课程
- **WHEN** 用户点击课程卡片
- **THEN** 跳转至对应课程详情页，渲染该课程完整信息

#### Scenario: 用户访问不存在的课程
- **WHEN** 用户访问 `/courses/<不存在的id>`
- **THEN** 返回 404 友好提示页面

### Requirement: 项目列表页
系统 SHALL 在 `/projects` 路由提供项目列表页，网格卡片展示所有商务数据分析实战项目，每个项目卡片包含：项目名称、行业标签、数据规模、难度。

#### Scenario: 用户浏览项目列表
- **WHEN** 用户访问 `/projects`
- **THEN** 页面以卡片网格形式展示全部项目，每个项目可点击进入详情页

### Requirement: 项目详情页
系统 SHALL 在 `/projects/<project_id>` 路由提供项目完整说明，包含三大模块：项目说明（业务背景与分析目标）、数据集介绍（数据来源、字段说明、数据规模）、任务描述（具体分析任务列表）。

#### Scenario: 用户点击某个项目
- **WHEN** 用户点击项目卡片
- **THEN** 跳转至项目详情页，按"项目说明 / 数据集介绍 / 任务描述"三段式渲染

#### Scenario: 用户访问不存在的项目
- **WHEN** 用户访问 `/projects/<不存在的id>`
- **THEN** 返回 404 友好提示页面

### Requirement: 统一导航与页脚
系统 SHALL 在所有页面顶部渲染固定导航栏（首页 / 课程 / 项目 三个栏目），并在底部渲染页脚（版权信息、联系方式占位）。

#### Scenario: 用户在任意页面切换导航
- **WHEN** 用户点击导航栏中的"课程"链接
- **THEN** 页面跳转至 `/courses`，导航栏保持高亮当前栏目

### Requirement: 视觉风格
系统 SHALL 使用蓝白灰色调，主色 `#1E5BB8`（蓝），辅助色 `#F5F7FA`（浅灰背景），文字色 `#2C3E50`（深灰），整体风格简洁、专业、清爽。

#### Scenario: 页面加载完成
- **WHEN** 任何页面渲染完成
- **THEN** 视觉风格符合蓝白灰三色系，无杂乱配色，间距合理，字体清晰
